# Galería Interactiva

Esta aplicación permite agregar, seleccionar y eliminar imágenes dinámicamente utilizando JavaScript y manipulación del DOM.

## Funcionalidades

- Agregar imágenes desde una URL.
- Seleccionar una imagen (solo una a la vez).
- Eliminar la imagen seleccionada.
